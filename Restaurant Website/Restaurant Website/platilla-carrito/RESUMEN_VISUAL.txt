🔗 ENDPOINTS API INTEGRADOS
═══════════════════════════════════════════════════════════════════════════════

✅ GET /api/productos
   ├─ Cuándo: Al cargar index.html
   ├─ Retorna: Array de productos
   └─ Usado en: Carga dinámica

✅ GET /api/clientes?email=...
   ├─ Cuándo: Antes de crear pedido
   ├─ Retorna: Array de clientes
   └─ Usado en: Búsqueda de cliente existente

✅ POST /api/clientes
   ├─ Cuándo: Si cliente no existe
   ├─ Estructura: {nombre, apellido, email, celular, dirección, ...}
   └─ Retorna: {id_cliente, ...}

✅ POST /api/pedidos
   ├─ Cuándo: Al procesar compra
   ├─ Estructura: {
   │     id_cliente: Number,          // REQUERIDO
   │     metodo_pago: String,         // REQUERIDO
   │     productos: [                 // REQUERIDO
   │       {id_producto, cantidad, precio}
   │     ],
   │     descuento: Number,           // Opcional
   │     aumento: Number              // Opcional
   │   }
   └─ Retorna: {message: "...", id: Number}

🔄 FLUJO DE DATOS (Ejemplo Real)
═══════════════════════════════════════════════════════════════════════════════

USUARIO: Juan Pérez (nuevo cliente)
────────────────────────────────────────────────────────────────

1. index.html carga
   ├─ GET /api/productos
   │  ← [{id:1, nombre:"Burger", precio:10.50, ...}, ...]
   └─ Muestra 12 productos dinámicamente

2. Usuario agrega "Burger Deluxe" al carrito
   ├─ localStorage: {"carrito_compras": [{id:1, nombre:"...", ...}]}
   └─ Contador: 1

3. Usuario va a checkout
   └─ Llena formulario
      ├─ Nombres: "Juan"
      ├─ Apellidos: "Pérez"
      ├─ Email: "juan@example.com"
      ├─ Celular: "31234567890"
      ├─ Dirección: "Calle 10 #20-30"
      └─ Método: "contra-entrega"

4. Sistema busca cliente
   ├─ GET /api/clientes?email=juan@example.com
   │  ← [] (no existe)
   └─ Necesita crear cliente

5. Sistema crea cliente
   ├─ POST /api/clientes
   │  ├─ Body: {nombre: "Juan", apellido: "Pérez", ...}
   │  └─ Response: {id_cliente: 5, message: "Cliente creado"}
   └─ Obtiene: id_cliente = 5

6. Sistema crea pedido
   ├─ POST /api/pedidos
   │  ├─ Body: {
   │  │   id_cliente: 5,
   │  │   metodo_pago: "contra-entrega",
   │  │   productos: [{id_producto: 1, cantidad: 1, precio: 10.50}],
   │  │   descuento: 0,
   │  │   aumento: 0
   │  │ }
   │  └─ Response: {message: "Pedido creado", id: 123}
   └─ Obtiene: Pedido ID = 123

7. Sistema limpia y redirige
   ├─ localStorage: {} (vacío)
   ├─ sessionStorage: {ultimoPedido: {id: 123, ...}}
   └─ Redirige: thankyou.html

8. Usuario ve confirmación
   ├─ "¡Gracias por tu compra!"
   ├─ "Número de pedido: 123"
   └─ "Tiempo estimado: 30-45 minutos"
